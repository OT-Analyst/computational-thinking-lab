# Computational Thinking Lab

This repository documents my journey in learning how to think through problems using Python as the tool. Each exercise is broken down using my personal problem‑solving framework, inspired by the core principles of computational thinking (decomposition, pattern recognition, abstraction, and algorithm design).

The goal of this project is to make my thinking visible — not just the final code — by documenting:
- the problem in my own words
- the inputs and validation
- the process and pseudo code
- the final solution
- test cases
- a reflection on my thought process

Each exercise lives in its own folder and includes both the code and the written reasoning behind it.

This is a working lab, not a polished portfolio. It will grow and evolve as my thinking improves.
